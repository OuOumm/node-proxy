## Node-Proxy

1. 什么是node-proxy？
> 以node.js为基础，实现Nginx反向代理功能的一个小工具
2. 为什么要用node-proxy？
> 在某些情况下，Nginx无法自行配置，但是可以运行node.js的情况下，此时，node-proxy就派上用场了。


## 使用教程
1. node-proxy如何使用？
   1. 安装node.js
   2. git clone node-proxy
   3. 打开终端，cd node-proxy
   4. 运行 npm install
   5. 设置配置信息
   6. 运行 node app.js
2. node-proxy如何配置？
   请查看app.js内详细备注
3. node-proxy如何扩展？
   想怎么扩展怎么扩展，欢迎提交pr
4. node-proxy如何贡献？
   请直接提交pr或者issue
